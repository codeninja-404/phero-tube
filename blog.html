Var , Let , Const in JavaScript

Variables can be declared is JavaScript with three keywords and they are var , let and const.

  • var : var is a keyword that is used to declare variables in JS it is a Global scope and Function scope. var can be redeclare and reassign.
  • let : let is a block scope JS keyword . which also an be used to declare to variable . But let can not be redeclared like var but it can be reassigned. one can not access let outside of the block.
  • const : const is also a block scope. and it can not be reassigned or re declared in any way .
